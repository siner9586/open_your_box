# Privacy Policy

Open Your Box 的隐私原则是最小化、脱敏、本地优先和明确授权。

## 默认不保存原始扫描结果

公共 Demo 站点不保存原始扫描结果。报告导入默认在浏览器本地解析，不上传到服务器。

## Demo 数据

Demo 数据不包含真实个人或真实公司，只使用 `example.com`、`demo_user`、`d***@example.com`、`203.0.113.10`、`198.51.100.22`、`192.0.2.15` 等演示值。

## 敏感值脱敏规则

- 邮箱：`d***@example.com`
- IP：`203.0.113.*`
- Token：默认不保存明文，只显示必要的脱敏摘要。
- Banner：不保存原始 banner，只保留端口、服务、产品、版本、组织、地区、最近观测时间和修复建议。

## 用户主动保存报告

如果未来接入数据库，只保存用户主动确认保存的脱敏摘要、风险分、任务状态和复查时间。默认不保存明文凭证、泄露样本、暗网链接或可复现攻击路径。

## 第三方 API 调用

第三方 API 只能在本地或服务端通过 Secret 调用。调用前必须确认目标属于本人、自有资产或明确授权资产。
